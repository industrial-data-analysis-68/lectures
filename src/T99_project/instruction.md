# Data Analysis Project

- 20% of final grade
- Individual assignment
- In-Class Presentation Date: 8th February 2025, 4:00 PM
  - 10 minutes presentation + 5 minutes Q&A
- Deliverables:
  - Original Dataset (CSV or Excel) if possible to share
  - Jupyter Notebook or Python Script
  - Presentation Slides (PDF)

## Project Guidelines

1. **Dataset Selection**:
   - Choose a dataset relevant to your field of interest.
2. **Data Processing**:
   - Clean and preprocess the data using Pandas.
3. **Analysis and Visualization**:
   - Perform analysis to derive meaningful insights from the data.
   - Create visualizations to illustrate key insights using libraries like Matplotlib, Seaborn, or Plotly.
4. **Dashboard Creation**:
   - Create a dashboard using Looker Studio to present your findings interactively.

## Project Presentation

- Motivation:
  - What inspired you to choose this dataset?
  - What questions are you trying to answer?
- Dataset Overview:
  - Briefly describe the dataset and its source.
- Processing Steps:
  - Explain the data cleaning and preprocessing steps you took.
- Analysis and Findings:
  - Present your analysis and key findings.
- Visualizations:
  - Showcase the visualizations you created and explain their significance.
- Dashboard Demonstration:
  - Walk through your Looker Studio dashboard, highlighting its features.
